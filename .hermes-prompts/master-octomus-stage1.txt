=== STRATEGIE: 1 CLONE + WORKTREE-URI + 2 BATCH-URI + MERGE + CARGO CHECK ===

=== PAS 1: CLONE REPO + WORKTREE-URI ===
Clonează repo-ul O SINGURĂ DATĂ, apoi creează worktree-uri pentru fiecare agent.
Toți agenții lucrează pe același sandbox, în worktree-uri separate.

[bash]
if [ -d /tmp/repo/.git ] && cd /tmp/repo && git remote get-url origin | grep -q "AdrianTuci1/warp-new"; then
  cd /tmp/repo && git fetch origin && git checkout master && git pull origin master
else
  rm -rf /tmp/repo
  git clone https://x-access-token:__GH_TOKEN__@github.com/AdrianTuci1/warp-new.git /tmp/repo
  cd /tmp/repo
fi

# Curăță worktree-uri vechi (dacă există)
for wt in agent-1-delete agent-2-lib agent-3-ai agent-4-branding agent-5-cargo agent-6-simplify agent-7-crates; do

  rm -rf "$wt_dir"
  git branch -D "octomus/$wt" 2>/dev/null || true
done

# Creează worktree-uri — fiecare agent are propriul director + branch

[/bash]

La final trebuie să avem un build care trece cargo check.

Batch 1: A1, A2, A3, A4 în paralel (delete + lib + AI strip + branding)
Batch 2: A5, A6, A7 în paralel (cargo + simplify + crate cleanup)
Apoi merge + cargo check.

=== MERGE + CARGO CHECK ===

După ce toți 7 agenții au terminat, fă merge-ul din /tmp/repo (clone-ul principal, unde ai toate branch-urile):

[bash]
cd /tmp/repo
git fetch origin
git checkout master
git checkout -b octomus/stage1
for b in agent-1-delete agent-2-lib agent-3-ai agent-4-branding agent-5-cargo agent-6-simplify agent-7-crates; do
  if git merge "origin/octomus/$b" --no-edit 2>&1; then
    echo "Merged $b cleanly"
  else
    echo "CONFLICT in $b — resolving with --theirs..."
    git diff --name-only --diff-filter=U | while read f; do
      git checkout --theirs "$f" 2>/dev/null || true
    done
    git add .
    git diff --cached --quiet || git commit -m "merge $b: resolve conflicts"
  fi
done
[/bash]

[bash]
cd /tmp/repo && cargo check 2>&1 | tail -80
[/bash]

Dacă erori: citește fișierele, editează, re-rulează cargo check (max 20 iterații).

La final:
[bash]
cd /tmp/repo
if git diff --cached --quiet; then echo "clean"; else git add -A && git commit -m "octomus: post-merge fixes — stage 1"; fi
git push origin octomus/stage1
[/bash]

Raportează "✅ STAGE 1 COMPLETE — octomus/stage1 pushed, cargo check trece!"
Menționează explicit: "Acum rulează master-octomus-stage2.txt pentru finalizare."
