=== MERGE FINAL + CARGO CHECK ===

După ce toți agenții au terminat, fă merge-ul din /tmp/repo (clone-ul principal):

[bash]
cd /tmp/repo
git fetch origin
git checkout octomus/stage1
git checkout -b octomus/release
for b in agent-8-ci agent-9-final agent-10-telemetry agent-11-remote agent-12-store agent-14-voice agent-15-skills; do
  if git merge "origin/octomus/$b" --no-edit 2>&1; then
    echo "Merged $b cleanly"
  else
    echo "CONFLICT in $b — resolving with --theirs..."
    git diff --name-only --diff-filter=U | while read f; do
      git checkout --theirs "$f" 2>/dev/null || true
    done
    git add .
    git diff --cached --quiet || git commit -m "merge $b: resolve conflicts"
  fi
done
[/bash]

[bash]
cd /tmp/repo && cargo check 2>&1 | tail -80
[/bash]

Dacă erori: citește fișierele, editează, re-rulează cargo check (max 20 iterații).

La final:
[bash]
cd /tmp/repo
if git diff --cached --quiet; then echo "clean"; else git add -A && git commit -m "octomus: post-merge fixes — stage 2"; fi
git push origin octomus/release
[/bash]

Raportează "🎉 OCTOMUS RELEASE READY — octomus/release pushed, cargo check trece!"

=== CELE 4 DEPENDENȚE CLOUD ÎNLOCUITE ===

După stage 2, Octomus are toate cele 4 dependențe cloud rezolvate local:

1. SSE event streaming     → LocalEventStreamer (file-based JSONL polling, 500ms)
2. Snapshot upload (GCS)   → agent-store/<conversation>/<agent>/snapshot/
3. Artifact reporting      → AgentStore::store_artifact() → artifacts/
4. Handoff/fork            → copiere locală conversație + snapshot
